What?

Image upscaling enlarges images by increasing a low resolution image's size to a higher resolution without losing image quality. For example, upscaling a 600 x 400 pixel image will produce a 2400 x 1600 pixel image without blurring it.

button »

How?

Our Cloud API isn't a GAN-based solution, thus it uses an advanced CV algorithm to enhance image quality when upscaling that doesn't generate new information in the upscaled image that doesn't already exist in the original image.
